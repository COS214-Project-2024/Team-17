Detailed Description

Header file for the HealthPolicy class, defining health-oriented budget allocation.

This file defines the HealthPolicy class, which implements a budget allocation strategy focused on health. The policy allocates a significant portion of the budget to health, with remaining funds divided among education and infrastructure. HealthPolicy integrates with the CityStructure to apply its unique budgeting approach in a health-prioritized manner.

Note
HealthPolicy serves as one of several budget policies available to the Government class for city resource distribution.
