Header file for the \mbox{\hyperlink{classBudgetPolicy}{Budget\+Policy}} class, defining the base interface for city budget policies. 

This file declares the \mbox{\hyperlink{classBudgetPolicy}{Budget\+Policy}} class, an abstract base class for creating budget allocation policies. Derived classes implement specific strategies for allocating a budget across different city departments, allowing flexible policy implementations. \begin{DoxySeeAlso}{See also}
