\doxysubsection{Detailed Description}
Factory class for creating commercial buildings. 

The \mbox{\hyperlink{classFactCommercial}{Fact\+Commercial}} class is a concrete implementation of the \mbox{\hyperlink{classFactoryBuilding}{Factory\+Building}} interface that provides methods to create various types of commercial buildings, such as malls, shops, and offices.
