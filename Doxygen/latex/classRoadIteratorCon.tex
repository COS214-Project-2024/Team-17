\doxysubsection{Detailed Description}
Concrete iterator for traversing a collection of road components. 

The \mbox{\hyperlink{classRoadIteratorCon}{Road\+Iterator\+Con}} class provides an implementation of the \mbox{\hyperlink{classRoadIterator}{Road\+Iterator}} interface, allowing for sequential access to the elements of a vector of \mbox{\hyperlink{classRoadComponent}{Road\+Component}} objects.
