\doxysubsection{Detailed Description}
Represents a main road component in the road network. 

The \mbox{\hyperlink{classMainRoads}{Main\+Roads}} class is derived from the \mbox{\hyperlink{classRoadComponent}{Road\+Component}} class and provides functionality for managing and interacting with main road segments, including displaying information, calculating traffic, and managing connections to other road components.
