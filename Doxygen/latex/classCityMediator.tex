\doxysubsection{Detailed Description}
Mediator class that facilitates communication between various city components. 

The \mbox{\hyperlink{classCityMediator}{City\+Mediator}} class acts as an intermediary, coordinating interactions between buildings, utilities, citizens, buses, and train stations. It ensures that all components can communicate and respond to changes within the city ecosystem.
