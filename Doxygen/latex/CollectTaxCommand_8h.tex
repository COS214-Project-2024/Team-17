\doxysubsection{Detailed Description}
Header file for the \mbox{\hyperlink{classCollectTaxCommand}{Collect\+Tax\+Command}} class, which encapsulates the tax collection command. 

This file defines the \mbox{\hyperlink{classCollectTaxCommand}{Collect\+Tax\+Command}} class, used to trigger the collection of taxes by the \mbox{\hyperlink{classGovernment}{Government}}. It implements the Command design pattern, allowing the \mbox{\hyperlink{classGovernment}{Government}} to collect taxes from all associated cities in response to a specific command execution. \begin{DoxyNote}{Note}
\mbox{\hyperlink{classCollectTaxCommand}{Collect\+Tax\+Command}} is part of a set of commands available to the \mbox{\hyperlink{classGovernment}{Government}} for various tax and budget actions. 
\end{DoxyNote}
