\doxysubsection{Detailed Description}
Abstract base class representing the state of a citizen. 

The \mbox{\hyperlink{classCitizenState}{Citizen\+State}} class serves as an interface for different states that a citizen can have. Derived classes should implement specific state behaviors and transitions.
