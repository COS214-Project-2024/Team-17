\doxysubsection{Detailed Description}
Factory class for creating landmark buildings. 

The \mbox{\hyperlink{classFactLandmarks}{Fact\+Landmarks}} class is a concrete implementation of the \mbox{\hyperlink{classFactoryBuilding}{Factory\+Building}} interface that provides methods to create various types of landmark buildings, such as parks, monuments, and community centers.
