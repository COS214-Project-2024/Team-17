\doxysubsection{Detailed Description}
Factory for creating residential buildings. 

The \mbox{\hyperlink{classFactResidential}{Fact\+Residential}} class inherits from the \mbox{\hyperlink{classFactoryBuilding}{Factory\+Building}} class and provides methods to create various types of residential buildings such as houses, townhouses, flats, and estates.
