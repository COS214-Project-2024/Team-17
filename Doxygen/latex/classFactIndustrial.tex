\doxysubsection{Detailed Description}
Factory class for creating industrial buildings. 

The \mbox{\hyperlink{classFactIndustrial}{Fact\+Industrial}} class is a concrete implementation of the \mbox{\hyperlink{classFactoryBuilding}{Factory\+Building}} interface that provides methods to create various types of industrial buildings, such as factories, warehouses, and plants.
