\doxysubsection{Detailed Description}
Abstract base class for building factories. 

The \mbox{\hyperlink{classFactoryBuilding}{Factory\+Building}} class serves as an interface for creating various types of buildings, including residential, commercial, industrial, landmarks, and service buildings. Derived classes must implement the specified methods to provide concrete building creation functionality.
